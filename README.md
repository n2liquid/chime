Description
-----------
chime is a chip tune sound library based on [T'SoundSystem](https://github.com/toyoshim/tss/) which provides MML based music system.

How to build
------------

### Preparation
You need node.js and npm.

````
$ git clone git://github.com/toyoshim/chime.git
$ cd chime
$ make tools
$ make build
````
